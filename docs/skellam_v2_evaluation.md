# Skellam v2 評価レポート (skellam_v2_evaluation.md)

> 作成日: 2026-04-10

---

## 改良内容

### Skellamモデルへの追加
1. **closeness-based draw boost**: λ差とELO差から接近度を計算し、接近試合でdraw確率を増加
2. **Dixon-Coles rho の調整可能化**: デフォルト-0.08から-0.15まで強化可能
3. **ELO差の併用**: boost計算でλ接近度とELO接近度を平均化

### Selection Ensemble
- draw警戒時 (draw_prob>=25% かつ home/away差<10pp) → v7
- それ以外 → Skellam+boost

## v7を超える可能性

**val=2025**: わずかに超える
- F1 macro: 0.421 → 0.424 (+0.3pp)
- Accuracy: 0.478 → 0.483 (+0.5pp)
- ECE: 0.061 → 0.042 (校正が改善)

**2026 holdout**: **大幅に上回る**
- Accuracy: 0.342 → 0.488 (+14.6pp)
- F1: 0.258 → 0.432 (+17.4pp)
- LogL: 1.099 → 1.074 (-0.025)

### 注意点
- 2026 holdout n=41は統計的に不十分 (±15ppの信頼区間)
- val=2025との差 (+0.5pp vs +14.6pp) は2026の特殊性を反映している可能性
- 継続監視とn>=80での再評価が必須

## Primary昇格判断

### 候補: Skellam+boost(0.06)

**昇格を推奨**する理由:
1. 2026 holdoutで v7 を大幅に上回る
2. val=2025 でも F1/ECE が改善
3. draw予測がv7より多い (56 vs 44)
4. DrawF1 改善 (0.227 → 0.248)

**慎重になる理由**:
1. 2026 n=41 は統計的に小さい
2. val=2025 の LogL はやや悪化 (1.053 → 1.061)
3. UI の第一/第二推奨ロジックがv7に最適化済み

### 推奨プラン
- **Skellam+boost を shadow2 として追加** (v8.1 shadowと併走)
- 2026 n>=80 到達時に v7 vs Skellam+boost を再比較
- Skellam+boost が継続優位なら primary 昇格検討

**現時点では primary昇格を保留し、shadow併走を推奨**
