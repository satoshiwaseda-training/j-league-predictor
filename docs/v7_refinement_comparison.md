## v7 → v7 refined 変更内容

## 改善の解釈

- bias_draw -0.82→-0.60: drawのベースロジットを上げ、draw予測の閾値を下げた
- scale_draw 1.01→0.80: closeness感度を下げ、一定のdraw確率ベースを確保
- 結果: draw=44→53に増加、draw precision 31.8%→35.8%に改善
