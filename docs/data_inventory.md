[======= Train =======][==== Validation ====][==== Holdout ====]

## 6. 取得方法

```python
from data_fetcher import get_historical_results, get_multi_season_results

# 単年取得
results_2024 = get_historical_results(2024, "j1")
results_2025 = get_historical_results(2025, "j1")

# 複数年一括
all_results = get_multi_season_results([2024, 2025], "j1")
# → 2024 + 2025 + 2026(現シーズン) を日付順で返す
```

## 7. 各フィールド仕様

| フィールド | 型 | 説明 | 例 |
|-----------|------|------|------|
| date | str | 試合日 (YYYY-MM-DD) | "2024-02-23" |
| home | str | ホームチーム (フル名) | "サンフレッチェ広島" |
| away | str | アウェイチーム (フル名) | "浦和レッズ" |
| home_score | int | ホームスコア | 2 |
| away_score | int | アウェイスコア | 0 |
| score | str | スコア文字列 | "2-0" |
| winner | str | 結果ラベル | "home" / "draw" / "away" |
| section | int | 節番号 | 1 |
| season | int | シーズン年 | 2024 |
| division | str | リーグ | "J1" |
