## 4. モデルが信頼できる条件

### 高信頼度
- max_prob >= 50% (n=48, acc=58.3%)
- ELO差が大きい (上位vs下位)
- home予測 (precision 54.8%)

### 中信頼度
- max_prob 40-50% (n=149, acc=54.4%)
- draw確率 25-30%帯 (acc=53.6%)

### 低信頼度(注意必要)
- max_prob < 40% (n=180, acc=41.7%)
- draw予測 (precision 35.8% — 3回に1回しか当たらない)
- 昇格組/新チーム (ELO蓄積不足)
- シーズン序盤 (特徴量が不安定)
