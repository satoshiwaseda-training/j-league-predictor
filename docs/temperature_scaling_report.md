## 判断: temp=1.0 を維持

理由:
1. F1 macro 0.425 が全temp中最良
2. draw# 44 が最多 (draw復活の目的に合致)
3. logL差は0.003 (統計的に無意味)
4. temperatureはモデル全体の確信度を操作するだけで、予測構造自体は変わらない

## 将来検討
- temp < 1 は確率をシャープにしlogL/Brierを改善するが、draw予測を抑制する
- これは「drawの確率は正しいが、argmaxではhome/awayが選ばれやすくなる」効果
- draw予測を維持しつつlogL改善するには、drawロジット自体の精度向上が必要
