- 節6-7 (n=9-10) で40-44% → val=2025(48.8%)より低いが2026序盤の特性

- 2026ではdraw=2件のみ (val=2025では53件)
- 原因: 2026序盤はELO蓄積が浅く、実力差が大きく見える → closenessが低い
- シーズン進行でELO精度向上 → draw予測増加の見込み
